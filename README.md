# ChordPosition

ギターのコードネームを入力すると、指板上の全ポジション（1〜6弦、0〜11フレット）にコード構成音を表示するWebアプリです。

## Live Demo

https://aike.github.io/ChordPosition/

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルド不要・依存ライブラリなし。

1. テキストボックスにコードネームを入力（例: `Am7`、`Cmaj7`、`F#m7b5`）
2. **表示** ボタンを押すか Enter キーを押す（入力のたびにリアルタイム更新）
3. 指板上にコード構成音が表示される

クイックボタンからよく使うコードをワンクリックで選択することもできます。

## 表示内容

| 表示 | 意味 |
|------|------|
| 赤い塗りつぶし円 | ルート音 |
| 青い枠線円 | その他の構成音 |
| 青い縦長の四角 | ポジションマーク（3・5・7・9フレット）|

ヘッダー部分にコード名・構成音名・インターバル（R / M3 / 5th / ♭7 など）を表示します。

## 対応コードタイプ

| 表記例 | コードタイプ |
|--------|------------|
| `C` | メジャー |
| `Am` / `Amin` | マイナー |
| `G7` | ドミナント7th |
| `Cmaj7` / `CM7` / `CΔ7` | メジャー7th |
| `Dm7` | マイナー7th |
| `Bdim` / `B°` | ディミニッシュ |
| `Baug` / `B+` | オーギュメント |
| `Dsus4` / `Dsus2` | サスペンデッド |
| `Cadd9` | アド9th |
| `C#m7b5` / `C#ø` | ハーフディミニッシュ |
| `Dm6` / `D6` | 6th |
| `G9` / `G11` / `G13` | テンションコード |
| `CmM7` | マイナーメジャー7th |
| `B°7` / `Bdim7` | ディミニッシュ7th |

シャープ（`C#`）・フラット（`Bb`）両対応。スラッシュコード（`C/E`）はベース音を無視してコードのみ表示。

## 動作環境

モダンブラウザ（Chrome / Firefox / Safari / Edge）。ビルドツール・フレームワーク不要。
